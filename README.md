
# CyberCamera

CyberCamera is a Nintendo 3DS homebrew camera app with a cyber/Y2K inspired control interface, real-time filters, photo capture, gallery browsing, timer capture, 2x BMP export.

The app focuses on keeping the top screen clean for the live camera preview while placing controls and status information on the bottom screen.

## Features

- Live 3DS camera preview
- Clean top-screen camera view
- Bottom-screen cyber-style control panel
- Photo capture to SD card
- PC-viewable BMP export
- Internal gallery viewer
- Auto-created capture folder
- Real-time filters
- Timer capture: 0 / 3 / 5 / 10 seconds
- 1x or 2x save scale
- Experimental anaglyph 3D photo mode
- 3D slider support for live stereo preview
- Settings saved to SD card

## Controls

| Button | Action |
|---|---|
| A | Capture photo |
| B | Open/close gallery |
| L / R | Change filter |
| X | Change timer |
| Y | Toggle HUD |
| SELECT | Switch front / outer camera |
| D-Pad Up | Toggle anaglyph photo mode |
| D-Pad Down | Toggle save scale 1x / 2x |
| START | Exit |

Gallery controls:

| Button | Action |
|---|---|
| L / R | Previous / next photo |
| Y | Delete selected photo |
| B | Return to camera |
